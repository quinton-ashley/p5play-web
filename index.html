p5play is a 2D game engine made by Computer Science teachers to help you turn your ideas into JavaScript code!

It uses p5.js or q5js for graphics rendering. We share the Processing Foundation's goal of make coding more accessible for artists, designers, educators, and beginners. Behind the scenes p5play v3 uses planck (Box2D), a physics simulator that's been used in popular 2D games like Tiny Wings and Angry Birds.

Ready to learn how to develop your own games or interactive art? Start by reading the "Learn" reference pages!
